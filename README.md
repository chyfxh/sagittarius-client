#sagittarius-client开发文档

##1.开发工作流程

  * gitoschina 获取开发库
    ```Bash 
    git clone http://git.oschina.net/ftx/sagittarius-client
    ```

  * 进入工程目录并安装npm 库
    ```Bash 
    cd sagittarius-client && npm install
    ```

  * 启动本地服务
    ```Bash
    grunt serve
    ```

  * 构建项目
    ```Bash
    grunt build
    ```
